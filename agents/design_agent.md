# Design Agent

## 역할

UI/UX 설계, 비주얼 디자인 방향, 컴포넌트 시스템.

## 담당 영역

- 페이지별 UI 설계
- 컴포넌트 시스템 (Badge, Button, Card, Tag)
- Tailwind CSS v4 커스텀 토큰
- 분석 결과 시각화 (MusicProfileCard, MusicDirectionSummary)
- 앨범 커버 목업 (AlbumMockup.tsx)
- 이미지 업로드 UI (드래그 앤 드롭 영역)

## 디자인 원칙

- **어두운 톤**: 음악 스튜디오/크리에이티브 환경에 맞는 다크 테마
- **정보 밀도**: MusicProfile의 9개 필드를 태그 형식으로 표시
- **이미지 강조**: Pollinations.ai 생성 앨범 커버를 결과 상단에 배치
- **공유 친화적**: 결과 페이지를 공유 링크로 그대로 접근 가능

## 페이지 목록

| 페이지 | 경로 | 설명 |
|--------|------|------|
| 홈 | `/` | 서비스 소개 |
| 분석 | `/analyze` | 텍스트/이미지 입력 + 결과 표시 |
| 결과 | `/result/[id]` | 분석 결과 상세 |
| 히스토리 | `/history` | 내 분석 목록 |
| 공유 | `/shared/[id]` | 공유 링크 결과 |
| 로그인 | `/login` | 이메일/비밀번호 로그인 |
| 회원가입 | `/signup` | 이메일/비밀번호 회원가입 |
| 관리자 | `/admin` | 통계 대시보드 |
